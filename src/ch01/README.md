나열된 수에서 최솟값과 최댓값 구하기

문제 정의 

1. 여러 개의 수가 배열에 있을 때 그 중 가장 큰 값과 가장 작은 값을 찾는다.
2. 배열의 몇 번째에 있는지 순서를 찾는다.
3. 반복문을 한번만 사용하여 문제를 해결한다.
4. 수의 예 : [ 10, 55, 23, 2, 79, 101, 16, 82, 30, 45 ]


해결하기
배열에 있는 수 중 맨 처음에 있는 값을 max와 min으로 가정하고, 
배열의 마지막 숫자까지 비교하면서 더 큰 수나 더 작은 수가 나올 때 max와 min의 값을 바꾼다.

추가로 그 때의 위치를 변수에 저장한다.
